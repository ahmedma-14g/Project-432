To-Do List Application
A simple To-Do List application built using Python and Tkinter, allowing users to manage their tasks by adding, deleting, searching, sorting, and assigning priorities to them.

Features
Add Task: Allows users to add tasks with descriptions and priorities (High, Medium, Low).
Delete Task: Delete tasks from the list.
Search Tasks: Search for tasks based on description or priority.
Sort Tasks by Priority: Organize tasks by their priority level.
Dynamic Display: Task list updates automatically as tasks are added, deleted, or sorted.
Requirements
Python 3.x
Tkinter (typically comes pre-installed with Python)
Installation
Make sure you have Python 3.x installed on your system.

If Tkinter is not already installed, you can install it using the following command:

pip install tk
Download or clone this repository to your local machine.

Usage
Run the todo_list_application.py script.
The application will open a window where you can:
Add tasks with the Add button.
Delete selected tasks with the Delete button.
Search for tasks using the Search function.
Sort tasks based on priority by clicking the Sort by Priority button.
Running the Application
To run the application, use the following command:

python todo_list_application.py
